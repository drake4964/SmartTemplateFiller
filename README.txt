# SmartTemplateFiller

JavaFX project using Gradle and Java 17.

## Run Instructions

1. Make sure Java 17 is installed and JAVA_HOME is set.
2. Use Gradle 8.5 or the included wrapper.
3. Run with JavaFX module path if needed:
   --module-path /path/to/javafx-sdk-17/lib --add-modules javafx.controls,javafx.fxml
