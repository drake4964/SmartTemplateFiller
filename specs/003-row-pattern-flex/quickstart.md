# Quickstart: Row Pattern Flex

No external dependencies or services to spin up.

To test this feature locally:

1. Setup project according to normal build procedures.
2. Build via Gradle: `./gradlew build`
3. Run the JavaFX app: `./gradlew run`
4. Load a CMM result file (e.g., `.asc`).
5. Double click a column to define mappings in Teach Mode.
6. Verify the UI presents 'Start Field', 'Fill Field', and 'Space Field' in Pattern Mode.
7. Test configuring invalid configurations (e.g., Fill = 0).
8. Verify export outputs match expected patterns.
